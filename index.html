⚙️ 設定雙人即時共編
使用免費的 JSONBin.io 雲端資料庫，兩人開同一個 HTML 就能即時同步打勾與分帳。
步驟一：前往 jsonbin.io 免費註冊
步驟二：登入後點右上角帳號 → API Keys → 複製 Secret Key（$2a$10$ 開頭）
步驟三：貼到下方，點「建立共享資料庫」
步驟四：把這個 HTML 檔傳給品郡，她填入同樣的 Key + Bin ID 就完成！
